# 游戏玩法设想

## "重生"系统

人物可以与其他N个人物"合体"（炼化）的方式，进行重生：
- 需要选择"主体"
    - 主体会记录自己已经重生了几次
    - 主体的部分特质（属性、知识、等等）会被保留（保留的多少根据重生次数来定）
    - 主体的技能会产生变异：
        - 允许保留原先的某几个技能
        - 保留的技能等级，根据重生次数来定（正相关）
        - 随机生成新的技能，新技能根据其他"非主体"的技能来决定
- 目的：
    - 消化多余的人物角色
    - 在有技能上限（或者将来的寿命上限）前提下，给老玩家以持续领先的机会
    - 对新玩家来说，在重生阶段是反击老玩家的机会
